个人博客搭建教程-第一期

前言

这篇文章会教你如何搭建一个属于你自己的博客网站

注: 方法只适用于安卓、Windows、Linux、Macos、苹果手机请移步

注: 安卓手机需要安装termux，使用Linux的方法进行安装

安装

第一步-安装nodejs,hexo,git

首先要安装nodejs
Windows可以去淘宝npm镜像站下载,速度比较快,目前最新17.3.0
Linux可以打开终端输入apt install nodejs进行安装
安卓和Linux一样打开termux输入apt install nodejs
